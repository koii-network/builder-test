# String Reversal Function

## Overview
This module provides a simple function to reverse a string in Python.

## Function: `reverse_string(s)`
- Input: A string `s`
- Output: The reversed string
- Method: Uses Python's slice notation `[::-1]` for efficient string reversal

### Example
```python
original = "Hello, World!"
reversed = reverse_string(original)
# reversed will be "!dlroW ,olleH"
```

## How to Run
Execute the script directly to see a demonstration:
```
python string_reversal.py
```
